MoneyMind 隐私政策

最后更新：2026年7月13日

1. 适用范围

本政策说明 MoneyMind 如何处理您的信息。MoneyMind 不要求注册账号，不接入银行卡，不使用广告、第三方分析或跨 App 行为追踪。

账本默认保存在设备本地。联网 AI 默认关闭；只有在您选择服务商、填写自己的 API Key、阅读说明并明确授权后，您主动使用 AI 功能所需的数据才会从设备直接发送给该服务商。

2. 设备本地数据

以下信息默认保存在您的设备和 MoneyMind App Group 容器中：

  • 交易记录，包括日期、类型、分类、金额和备注
  • 预算、定期账单、分类纠正记录和截图防重复指纹
  • 货币、主题、提醒、AI 授权状态等应用设置

您主动创建的 CSV 或备份文件由您选择保存位置并自行管理。通过小组件、Siri 或快捷指令产生的数据只在您的 Apple 设备和 MoneyMind 容器之间传递。

3. 麦克风、语音与截图

  • 麦克风和 Apple 设备端语音识别仅在您主动使用语音记账时启用；原始录音不会保存在账本中，也不会发送给大模型服务商。
  • MoneyMind 只接收您通过系统照片选择器明确选中的图片，不申请读取整个照片图库。
  • “轻点背面记账”需要您预先创建并绑定系统快捷指令；只有在您触发快捷指令时，系统才会截取当前屏幕并把该图片交给 MoneyMind。
  • 截图和语音识别失败时，可以继续使用本地识别或手动记账。

4. 使用自己的云端 AI（BYOK）

MoneyMind 2.1 不提供或中转官方 AI 网关。联网 AI 使用您自己的服务商账号、API Key 和额度，请求从设备通过 HTTPS 直接发送给您选择的服务商。API Key 保存在 iOS 钥匙串中，不写入账本或 MoneyMind 导出的备份文件。

在您明确授权并主动使用相应功能时，服务商可能收到：

  • 语音转写文字、您选择或通过快捷指令提交的截图、截图 OCR 文字，以及近期分类纠正记录，用于识别记账信息
  • 您的问题、月度汇总，以及最多 40 笔近期账单的日期、分类、金额和备注，用于账本问答或消费洞察
  • 您的 API Key，作为请求鉴权信息；服务商可据此把请求关联到您的服务商账号

不同服务商的数据保存地点、期限、模型改进设置和删除方式由该服务商及您的服务商账号设置决定。MoneyMind 开发者不经由自有服务器接收或保存这些请求正文。更换服务商或切换连接模式后，MoneyMind 会要求您重新授权。

5. 可选 AI 服务商

MoneyMind 当前支持以下自带 Key 的服务商。使用前请阅读您选择的服务商政策：

  • 智谱 AI：隐私政策
  • DeepSeek：隐私政策
  • OpenAI：隐私政策
  • Google Gemini：隐私政策

某些服务商可能在您所在国家或地区之外处理数据。请勿在 AI 输入或截图中提交完成记账功能不需要的个人信息。

6. App 内购买

Pro 购买和订阅由 Apple App Store 处理。MoneyMind 使用 StoreKit 在设备上读取已验证的购买权益，不接收您的支付卡号或银行账户信息。年度订阅可在 Apple 账户的订阅设置中管理或取消。

7. 数据用途与追踪

发送给您所选 AI 服务商的财务信息、用户内容和图片，仅用于完成您请求的记账识别、账本问答或个性化消费洞察。由于 API Key 对应您的服务商账号，这些数据可能与该账号关联。

MoneyMind 不出售或出租数据，不展示广告，不把数据用于广告画像，也不进行跨 App 或跨网站追踪。

8. 安全与您的选择

  • 本地数据位于 iOS 沙盒和 App Group 容器中，并受系统安全机制保护。
  • 联网 AI 请求使用 HTTPS；API Key 存储在 iOS 钥匙串中。
  • 您可以拒绝麦克风、语音识别和通知权限，继续使用其他功能。
  • 您可以在“AI 智能”中撤回当前服务商授权或清空 API Key，撤回后继续使用本地记账和本地识别。

9. 导出与删除

您可以在设置中导出 CSV 或完整备份，也可以删除单笔交易或清除账本、预算、定期账单及本地 AI 辅助记录。您另行保存的导出文件需要自行删除。

卸载 MoneyMind 会删除 App 容器内的数据。为了确保服务商侧数据和 API Key 按您的要求处理，请同时使用服务商账号提供的管理或删除入口，并在卸载前于 MoneyMind 中清空 Key。

10. 儿童隐私

MoneyMind 不面向儿童投放广告或建立画像。未成年人启用联网 AI 前，应由监护人阅读本政策并决定是否授权。

我们可能随功能和法律要求更新本政策，并在本页面标注新的更新日期。
